# hauntedps1-2020
This is a project for the [Haunted PS1 Summer of Screams game jam](https://itch.io/jam/haunted-ps1-summer-spooks).

Voting process is [held on twitter](https://twitter.com/HauntedPs1)

## Tools

* Unity 2019.4.3f1 (LTS)
* Blender 2.79b
